Linux snap-Pakete¶

Snap (wiki) ist ein Paket-Manager. Mit dem Paket-Manager können Pakete unabhängig vom nativem Paket-Manager einer Linux Distribution installiert werden. Snap kann auf den gängigen Distributuionen installiert werden (s.a. install snapd).
