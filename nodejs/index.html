Node.js (development)¶

install: Node.js / VSCode

$ sudo -H ./nodejs-dev.sh install all

Für die Entwicklung mit Node.js wird eine globale Installation benötigt, dabei empfiehlt sich die in Kapitel NodeSource: Node.js Binary Distributions beschriebene Installation. Die lokale Installation meint immer die Installation in ein Node.js Projekt. Die globale Installation erfolgt immer in das OS (oder alternativ in den $HOME Ordner). Als Editor für die Node.js Projekte eignet sich VSCode, dessen empfohlene Installation im Kapitel Installation VSCode Paketquellen beschrieben ist.
