Samsung Exynos 7 Octa 7870 (Joshua)¶

Quelle: PDADB Samsung Exynos 7 Octa 7870 (Joshua) datasheet

  • Design by: Samsung

  • Type: Exynos 7 Octa 7870

  • Hersteller: Samsung

  • Codename: Joshua

  • Released: 2016

  • Funktion: Multi-core Application Processor with Modem

Architektur¶

  • ALU Bitlänge: 64 bit

  • Instruction Set: ARMv8-A (A32, A64)

  • Prozessor Kerne: 8

  • Typ: 8x ARM Cortex-A53 MPcore

Datenbus¶

  • Memory Interface(s): mobile (LP) DDR2 SDRAM, DDR3L (LV) SDRAM, mobile (LP) DDR3 SDRAM

  • Datenbus Breite: 32 bit

  • Datenbus Kanäle: 1 ch

Taktfrequenz¶

  • 400 MHz min.

  • 1590 MHz max.

Cache Speicher¶

Technologie und Packung:

  • min. Packungsdichte: 14 nm

  • Halbleiter Technologie: CMOS

Sonstige Features:

  • Cellular: CSD (GSM) 9.6 kbps, GPRS, EDGE (EGPRS), UMTS 384 kbps (W-CDMA), HSDPA (3.5G UMTS downlink protocol), TD-SCDMA (UMTS 3G in China), LTE 300 Mbps / 50 Mbps (Cat. 6)

  • Embedded GPU: ARM Mali-T830 GPU

  • Number of GPU cores: 2-core GPU

  • GPU Clock: 772 MHz

  • Special Features: Quad 1.6 GHz ARM Cortex-A53 Harvard Superscalar processor cores + Quad ARM Cortex-A53 Harvard Superscalar processor cores, HMP, ARM big.LITTLE architecture, 64/32-bit Multi-layer AHB/AXI bus, ARM TrustZone, ARM NEON SIMD engine, LPDDR2 / LPDDR3 / DDR3L SDRAM interface, embedded GSM / GPRS / EDGE / TD-SCDMA / UMTS / HSPA+ / FDD-LTE / TDD-LTE Cat. 6 (CA) 300 Mbps cellular modem, NAND flash, moviNAND, SATA, embedded GPS module with GLONASS, 1080p 60 fps video encode/decode, dual ISP

Global Positioning:

  • GPS

  • GLONASS (Russisches Protokoll)
