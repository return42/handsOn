Archivierung / Komprimierung¶

Auf Linux ist so ziemlich jeder Packer verfügbar, hier eine kleine Auswahl:

  • file-roller GUI archive manager for GNOME, siehe GNOME File Roller

  • tar The GNU version of the tar archiving utility, siehe Manual tar

  • zip und unzip siehe Manual zip und unzip

  • gzip GNU compression utilities, siehe Manual gzip

  • rar und unrar siehe Manual rar und unrar-free

  • p7zip-full 7z und 7za Archiver, siehe Manual 7z und 7za

  • p7zip-rar non-free rar Bibliothek für p7zip-rar

  • cabextract Microsoft Cabinet file unpacker, siehe Manual cabextract

  • sharutils siehe Manual shar, unshar, uuencode, uudecode

  • uudeview Smart multi-file multi-part decoder siehe Manual uudeview

  • mpack pack a file in MIME format , siehe Manual mpack

  • unace siehe Manual unace

  • arj siehe Manual arj

Installation¶

$ sudo -H ./scripts/ubuntu_install_pkgs.sh archTools
