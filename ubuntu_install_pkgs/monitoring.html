Prozess-Monitoring¶

Zu den System Monitoring-Tools zählen:

  • htop: das schönere top http://hisham.hm/htop/

  • lm-sensors

  • iotop

  • fatrace report system wide file access events (fatrace)

Installation¶

$ sudo -H ./scripts/ubuntu_install_pkgs.sh monitoring
